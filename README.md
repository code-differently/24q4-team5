# 24q4-team5
Project repo for team 5

## What we will build

We will build a full-stack web application that is a basic Employee Management Application with CRUD features:

* Create Employee
* List Employee
* Update Employee
* Delete Employee
  
## Server-side tools and technologies used

* Spring Boot 3.4.0
* Maven
* JDK 1.7
* Embedded Tomcat 8.5+
* MySQL Database
